# AScan
基于python开发的主动漏洞扫描工具    
功能：    
1、支持子域名字典枚举扫描    
2、支持目录字典枚举扫描    
3、支持主动请求漏洞探测扫描    
3、支持弱口令字典爆破扫描    

运行：python Ascan.py
